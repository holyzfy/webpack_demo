# webpack_demo

## 安装

0. `npm install -g webpack sass-lint eslint validate_filename`
0. `npm install`

## 启动

运行`webpack --watch --process`，构建结果请看`dist`目录

## 预览

0. 运行
    ```
    webpack --output-public-path http://dev.f2e.tcredit.com:8021/webpack_demo/dist/
    ```
